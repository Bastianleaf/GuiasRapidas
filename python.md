# Resumen
Lenguaje de programación de alto nivel.
# Listas
### Crear lista vacía
`l = []`
### Agregar elemento
`l.append(x)`
### Obtener último elemento
`l[-1]` ó `l[len(l) - 1]`
# Diccionarios
### Crear diccionario vacío
`d = {}`
### Agregar elemento
`d[key] = value`
### Iterar sobre diccionario
`for key, value in d.items():`
# Funciones
### Expandir lista a argumentos
`foo(*l)` es equivalente a `foo(l[0], l[1], ...)`
